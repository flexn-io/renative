React Native Electron Next Engine

Overview

React native based engine with web transpiler provided by react-native-web and render by nextjs and native desktop wrapper provided by electron

Supported Platforms

macos, windows

Required Plugins

react, react-art, react-dom, react-native, react-native-web, next

Required NPM Packages

macos (dependencies)

  • html-webpack-harddisk-plugin, html-webpack-plugin, @zeit/next-css, next, next-routes, @expo/next-adapter, next-compose-plugins, next-fonts, next-images, babel-preset-expo, next-transpile-modules macos (devDependencies)
  • electron, electron-builder

windows (dependencies)

  • html-webpack-harddisk-plugin, html-webpack-plugin, @zeit/next-css, next, next-routes, @expo/next-adapter, next-compose-plugins, next-fonts, next-images, babel-preset-expo, next-transpile-modules windows (devDependencies)
  • electron, electron-builder
