Firefox OS Platform

FirefoxOS

Requirements

  • FirefoxDeveloperEdition for IDE + Emulator

After installation you can launch it via Applications:

Tools -> Web Developer -> WebIDE

Run

Run on Simulator

rnv run -p forefoxos

Run on Device

rnv run -p forefoxos -d

Run in Browser

rnv run -p forefoxos --hosted
